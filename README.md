# Introduction to R Programming

This repository contains beginner-friendly learning materials for R programming.  
It is designed for students with no prior programming experience.

The content introduces fundamental concepts of R, with clear explanations and simple, practical examples. The aim is to help learners build confidence in R programming step by step.

---

## Who is this for?

This repository is suitable for:
- Absolute beginners to R
- Learners with no prior coding background
- Anyone looking for a gentle introduction to R and RStudio

---

## How to use this repository

- Files are written as learning notes with example R code
- You can read the materials directly on GitHub
- Beginners are encouraged to run the code line by line to understand the output

---

## Requirements

- R (latest version recommended)
- RStudio (optional but strongly recommended)
